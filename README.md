# Quock
Thirst year project student, a base for a FPS online made with babylon.js

## Build with linux (ubuntu)

### 1. Install the dependencies:

	node.js

### 2. build the server

    npm install

### 3. Start the server:
	node server.js
 (you can duplicate the page to simulate other users)
